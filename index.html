Openpassword

What's OpenPassword?

Openpassword is a python API to access an Agilekeychain.

Can I see some API examples?

Currently OpenPassword is in early development stage, and most of the API is yet to implement. Here are some examples of the few implemented bits:

Unlocking the keychain

from openpassword import Keychain
from openpassword import EncryptionKeyFactory

keychain = Keychain(EncryptionKeyFactory(keys).key_for_security_level("SL5"))
keychain.unlock()

Decrypting an encryption key

from openpassword import EncryptionKey

encryption_key = EncryptionKeyFactory(keys).key_for_security_level("SL5")
decrypted_key = encryption_key.decrypt('masterpassword123')

The keys parameter passed to the EncryptionKeyFactory is a dictionary similar to the one found in the EncryptionKeyFactorySpec file.
